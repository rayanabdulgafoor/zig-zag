## Overview

Zig Zag is a stand-alone wooden skill game designed for single-player interaction. The game features a pivot-mounted zig-zag maze where players guide a ball through a curved path by manually tilting the platform using a control rod.

![Hero Image](/assets/zig_zag_render01.png)

The farther the ball travels without falling off the track, the higher the score earned by the player. The system was designed with interchangeable maze configurations and a fully parametric CAD workflow to support rapid customization and fabrication.

![Hero Image](/assets/zig_zag_render03.png)

The project combines parametric design, digital fabrication, CNC machining, and rapid prototyping techniques.

![Hero Image](/assets/zig_zag_render02.png)

## Game Mechanics

The player places a ball at the starting point and tilts the maze platform using a control rod to guide the ball through the zig-zag path.

Scoring zones are positioned at the far end of the track:

* Longer successful travel distance results in higher scores
* Each player receives 5 attempts
* Final scores are accumulated and ranked on a leaderboard

Key design parameters include:

* Thread-hole positioning
* Maze clearance
* Hole height and spacing
